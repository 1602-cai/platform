# 附录

## 行情数据字段列表

### tick - 分笔数据

*   time - int 时间戳
*   price - float 最新价
*   volume - int 成交量
*   amount - float 成交额
*   bid1 - float 买一价
*   ask1 - float 卖一价
*   bidvol1 - int 买一量
*   askvol1 - int 卖一量

### 1m / 5m / 1d - K线数据

*   time - int 时间戳
*   open - float 开盘价
*   high - float 最高价
*   low - float 最低价
*   close - float 收盘价
*   volume - int 成交量
*   amount - float 成交额
*   preClose - float 前收价（仅日线）
*   suspendFlag - int 停牌标记（仅日线）

### 除权数据

*   time - int 时间戳
*   cash - float 现金分红
*   share - float 送股
*   bonus - float 配股
*   spj - float 增发价格

### l2quote - level2实时行情快照

*   time - int 时间戳
*   lastPrice - float 最新价
*   volume - int 成交量
*   amount - float 成交额
*   bid1-10 - float 买一到买十价
*   ask1-10 - float 卖一到卖十价
*   bidvol1-10 - int 买一到买十量
*   askvol1-10 - int 卖一到卖十量

### l2order - level2逐笔委托

*   time - int 时间戳
*   price - float 委托价
*   volume - int 委托量
*   orderType - int 委托类型

### l2transaction - level2逐笔成交

*   time - int 时间戳
*   price - float 成交价
*   volume - int 成交量
*   buyNo - int 买方委托号
*   sellNo - int 卖方委托号

### l2quoteaux - level2实时行情补充（总买总卖）

*   time - int 时间戳
*   totalBidVol - int 总买量
*   totalAskVol - int 总卖量

### l2orderqueue - level2委买委卖一档委托队列

*   time - int 时间戳
*   bidQueue - list 买方委托队列
*   askQueue - list 卖方委托队列

## 数据字典

### 证券状态

*   0 - 正常
*   1 - 停牌
*   2 - 退市

### 委托类型

*   0 - 买
*   1 - 卖

### 委托方向

*   0 - 买
*   1 - 卖

### 成交标志

*   0 - 成交
*   1 - 撤单

### 现金替代标志

*   0 - 允许现金替代
*   1 - 不允许现金替代

## 代码示例

### 时间戳转换

```python
import datetime as dt

# 将时间戳转换为datetime
timestamp = 1640995200
dt_obj = dt.datetime.fromtimestamp(timestamp)
print(dt_obj)

# 将datetime转换为时间戳
dt_obj = dt.datetime(2022, 1, 1, 0, 0, 0)
timestamp = dt_obj.timestamp()
print(timestamp)
