# 基础行情信息

## 获取合约基础信息

    get_instrument_detail(stock_code, iscomplete=True)
    

*   释义
    *   获取合约基础信息
*   参数
    *   stock\_code - string 合约代码
    *   iscomplete - bool 是否获取完整字段
*   返回
    *   pd.DataFrame 数据集，数据字段详见附录 - 合约信息字段列表
*   备注
    *   无

## 获取合约类型

*   释义
    *   获取合约类型
*   参数
    *   stock\_list - list 合约代码列表
*   返回
    *   dict 数据集 { stock1 : type1, stock2 : type2, ... }
*   备注
    *   无
